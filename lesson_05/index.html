Властивість display

  • Для блочних тегів значення:
    • display: block
  • Для рядкових тегів значення:
    • display: inline
  • Для блочнj-рядкових тегів значення:
    • display: inline-block

Змінює тип тегу з БЛОЧНОГО на Рядковий, та РЯДКОВОГО на блочний

Для блочно-рядкового тегу значення BLOCK змусить наступний елемент розташовуватися з наступного рядка, а значення INLINE взагалі нічого не змінює.
